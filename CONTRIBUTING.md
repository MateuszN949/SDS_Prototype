# Contributing

Thank you for your interest in contributing.

## Workflow

1. Fork the repository.
2. Create a feature branch.
3. Preferably, follow the [conventional commit](https://www.conventionalcommits.org/en/v1.0.0/) convention.
4. Submit the contribution [form](https://put.poznan.pl/).
5. Open a pull request.

## Commit Convention

Examples:

feat: add batch prediction support

fix: correct evaluation metric

docs: update installation guide

## Pull Requests

All pull requests are reviewed before merging.
